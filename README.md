# Pong_WithPython
Pong foi o primeiro jogo eletrônico comercialmente bem sucedido e hoje é uma referência para a história dos jogos digitais, assim como para programadores. O jogo possui mecânicas simples e pode ser replicado em diversas linguagens de programação diferentes. Para este projeto, o jogo Pong foi desenvolvido utilizando a linguagem Python.

Para fazer o jogo funcionar, foi utilizada a biblioteca "turtle". Uma biblioteca muito divertida e com diversas funcionalidades, podendo ser utilizadas para criar desenhos vetorizados ou até mesmo jogos utilizando formas simples. 

O Pong possui uma mecânica simples, porém, ainda possui uma série de passos aos quais devemos nos atentar.

Passo 1: Devem existir dois jogadores e os dois devem poder se movimentar para cima e para baixo utilizando teclas diferentes

Passo 2: A bolinha deve sempre estar em movimento.

Passo 3: Ao colidir com um dos jogadores, a bolinha deve mudar sua direção. O mesmo vale para as paredes de cima e de baixo.

Passo 4: Deve-se ter um placar para quando a bolinha passar por um dos jogadores, um ponto ser marcado para o adversário.

Passo 5: Quando a bolinha reiniciar, deve ir em direção ao jogador que levou ponto na rodada passada.

Esses foram os passos principais para desenvolver a lógica por trás do jogo. Assim sendo, o jogo foi divido em classes de acordo com as funções necessárias, ou seja, Jogador, Bola e placar.
